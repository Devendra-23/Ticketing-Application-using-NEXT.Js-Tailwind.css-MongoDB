## Ticketing-App
- A simple ticketing application using features of Next.js, Tailwind.css, MongoDB & Mongoose to provide a smooth experience!
- CRUD functionality


## TECH-STACK
- Next.js
- Mongoose
- MongoDB
- Tailwind.css
